## 🖖 Introduction
[zoomkoding.dev](https://sad-haibt-6a7a35.netlify.app/)는 Gatsby와 React를 활용해서 만든 **개인 블로그**입니다.  
커스터마이징이 어려웠던 이전 jekyll theme를 벗어나, 나만의 theme으로 디자인과 개발을 해보는 중에 있습니다.
> 계속해서 **netlify**를 이용해서 배포하고 있습니다!
